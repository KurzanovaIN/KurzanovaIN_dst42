Задача проекта: отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, находящихся в группе риска.

И сделать это можно с помощью модели, которая предсказывала бы результаты госэкзамена по математике для каждого ученика школы. Провести разведывательный анализ данных и составить отчёт по его результатам. 

В качестве исходных данных используется датафрейм stud_math.csv

Этапы работы над проектом:

1. Проверка данных.
2. Проведена первичная обработка данных
3. Составили диаграммы распределение признака для числовых переменных, устранили выбросы.
4. Оценка количества уникальных значений для номинативных переменных
5. Проведен корреляционный анализ количественных переменных
6. Отбраны не коррелирующие переменные.
7. Проанализировали номинативные переменные и устранили те, которые не влияют на предсказываемую величину (в нашем случае — на переменную score).
8. Сформулированы выводы относительно качества данных и тех переменных, которые будут использоваться в дальнейшем при построении модели.
